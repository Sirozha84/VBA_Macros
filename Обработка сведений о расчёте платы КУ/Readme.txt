Обработка сведений о расчёте платы коммунальных услуг
Для работы скрипта требуется в книгу добавить страницы с услугами и справочник с адресами.
Далее данные соберутся во временную страницу Temp, сопоставятся данные со справочником и произойдёт
фильтрация данных.

Настройка

Этапы можно отключать закомментировав их вызов в процедуре Start
Но если отключить сбор данных "DataCollection", то нужно раскоментировать инициализацию переменных
и внести стартовые данные вручную (max - количество строк и tabs - количество столбцов)
