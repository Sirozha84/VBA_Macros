Mics

Библиотека с разными общими интерфейсными фишками (статусная строка, создание таблиц)
NewTab - создание новой страницы
SheetExist - Проверка на существование страницы
FindMax - Поиск последней строки
Progress - вывод информации о процессе
MethodSelect - выбор метода поиска (если сортированная - выбирается бинарный, быстрый поиск, если нет - обычный перебор)
Search - поиск по таблице
