Сравнение таблиц

Состыковывание двух таблиц сопоставляя строки по первому столбцу и делая различные действия с другими столбцами.
Запуск программы через модуль Starter, процедура Start

Версия 0.1 (20.04.2020)
* Сравнение двух таблиц, указанных в опциях
* Создание при желании результирующей таблицы
* Ручная настройка параметров шапки и количества колонок

Версия 0.2 (21.04.2020)
* Исправление поиска (не всегда находились значения если они в разных регистрах
* Исправлена статусная строка (программа рушилась, если количество строк переходило за 32 тысячи)

Версия 0.3 (22.04.2020)
* Автоматический выбор метода поиска (бинарный или перебором)
* Новое действие "Копировать" - копирует значения с обоих таблиц
* Новое действие "Сравнение" - сравнивает значения у двух таблиц и раскрашивает в зависимости от результата
* Разные раскраски у разных действий
* Оптимизация кода
