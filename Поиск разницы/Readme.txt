Сравнение двух и более таблиц на наличие новых и удалённых записей.
Поиск можно вести по сцепке нескольких ячеек идущих подряд.
Так же есть функция нахождения изменения у существующих записей в определённых ячейках.

Настройка поиска:

Const newTab = ""           'Новая таблица
Const fNew = 1              'Колонка в "Новой" таблице
Const oldTab = ""           'Старая таблица таблица (первая из нескольких, остальные добавляеются в процедуру NewAndOldFind
Const fOld = 1              'Колонка в "старых" таблицах
Const fields = 2            'Количество колонок для сравнени
Const maxTabs = 8           'Максимум полей

Сравнение записей:

Блок сравнения можно модефицировать в процедуре AddDead после комментария 'Сравнение
