Расчёт потребления по трём полям в разрезе дома
Общий итог по дому пишется в каждой строке

P.S.для СОИ

Пример:

1-2-3-4		11	12	13
----------------------
Адрес 1		1	0	2
Адрес 1		2	1	2
Адрес 2		1	0	0
Адрес 3		0	2	2

Считается как:

1-2-3-4		11	12	13		20
------------------------------
Адрес 1		1	0	2		8
Адрес 1		2	1	2		8
Адрес 2		1	0	0		1
Адрес 3		0	2	2		4
